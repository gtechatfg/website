Exercism Python Track

  • Signup on Exercism
  • Login and then Join the Exercism Python Track
  • Select "Mentored Mode"
  • Setup the exercism CLI tool on your machine
  • Complete 4 exercises
  • Post the screenshot after the 4 exercises were approved

Additional Info

    • FITA Python tutorial: https://www.fita.in/python-tutorial/
    • And many others ... Google It!
  • Create a learning group with other students and challenge each other
