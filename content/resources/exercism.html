Exercism Tracks

Exercism is an online community to help students to practice their skills by completing multiple exercises.

Exercism offers tracks for a wide variety of languages such as Python, C, C++, etc. Each track has a set of core exercises and a large set of additional exercises. Exercises are classified on the basis of difficulty as well as learning goals.

Students can complete exercises and for each completed solution you can gain points.

Each exercise comes with Instructions on the exercise. You can code your solution on your machine, and upload the soluion. Before uploading a solution, run the test cases to see if your code works at a basic level.

  • Signup on Exercism
  • Review the initial steps on Getting Started
  • Login and then Select a Track
  • Select "Mentored Mode"
  • Setup the exercism CLI tool on your machine
  • Complete an exercise
  • Once the exercise is approved, publish your solution to public.
  • Copy the link to the exercism exercise page
  • Copy the public link to your solution
  • Include the exercise link and the exercism link and post a message on #students channel on Discord. Use #exercism tag for your message

Additional Info

  • Try to create your own solutions for the problems
  • Create a learning group with other students and challenge each other to complete the exercises.

Example Message

Hi! I've completed the Pangram exercise on exercism. #exercism
Exercism: https://exercism.io/tracks/python/exercises/pangram
My solution link is: https://exercism.io/tracks/python/exercises/pangram/solutions/08120bdfcca641da95681e2a8cceac10

Points

Each Exercism exercise has a difficulty level denoted on the top left of the exercise page.

  • Easy Task: 1 Stars
  • Medium Task: 2 Stars
  • Hard Task: 3 Stars
